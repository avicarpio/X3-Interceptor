Controls, encara que es poden veure al video Gameplay:

W - Down
S - Up
A - Roll Left
D - Roll Right
Shift - Turbo
Space - Shoot



El nostre joc consisteix en destruir les naus enemigues, però s'ha d'anar amb compte ja que si col·lisiones amb una de les naus, perdràs la partida.

A part de els objectius obligatoris, dels opcionals hem fet:
- HUD
- Well implemented Bullet Manager
- More advanced Enemy movement (e.g. waypoints)
- Different types of enemy
- Rendering Optimisations: vam fer que les bales no tinguessin malla per tal d'estalviar recursos, i hem fet un control de bales per tant que no hi hagi saturacio i no es pugui estar disparant constantment
- Multimeshes / multimaterial
- Different shaders
- Speed Control
